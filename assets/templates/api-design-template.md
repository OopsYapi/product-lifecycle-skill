# 开放API设计模板

---

## API概述

- API版本：v1
- 基础路径：/api/v1
- 认证方式：OAuth 2.0 + API Key
- 数据格式：JSON
- 字符编码：UTF-8

---

## 通用规范

### 请求头

| Header | 必填 | 说明 |
|--------|------|------|
| Authorization | 是 | Bearer {token} |
| Content-Type | 是 | application/json |
| X-Request-Id | 否 | 请求追踪ID |

### 响应格式

```json
{
  "code": 0,
  "message": "success",
  "data": {},
  "timestamp": 1700000000
}
```

### 错误码

| 错误码 | 说明 | HTTP状态码 |
|--------|------|-----------|
| 0 | 成功 | 200 |
| 1001 | 参数错误 | 400 |
| 1002 | 认证失败 | 401 |
| 1003 | 权限不足 | 403 |
| 1004 | 资源不存在 | 404 |
| 2001 | 业务错误 | 400 |
| 5001 | 系统错误 | 500 |

## 安全机制

| 安全维度 | 措施 |
|---------|------|
| 传输安全 | HTTPS |
| 认证 | OAuth 2.0 + API Key |
| 授权 | RBAC |
| 限流 | 100次/分钟 |
| 审计日志 | 所有API调用记录 |
| 数据脱敏 | PII字段脱敏 |
